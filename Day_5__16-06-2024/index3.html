Margin and Padding - Faded out

Margin is the spacing applied to the outer part of the element while padding is the spacing applied to the inner part of the element.


Opacity

Opacity is the level of brightness you want to attain for a particular element.

Display

An element can be set to be displayed 'none' or 'block'

All elements have their display property set to 'block' by default.

What's the difference between opacity and display

Opacity still leaves the element in your html tree while display removes it completely.

Opacity will leave it in the html tree but blurs it out completely, meaning that the user won't be able to see the content.
