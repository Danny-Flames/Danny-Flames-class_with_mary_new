Flex Box

Css flex is mainly for the layout of elements on your page.

It takes care of how elements are dispayed horizontally and vertically.


The properties that work along side 'flex' are display, justify-content, align-items, align-self and so on...

The div element below with a background of yellow is a flex element.
Note that for an element to fully respect flex and its corresponding properties listed above, such element(parent element) must have a fixed height and width.
