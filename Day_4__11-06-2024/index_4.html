CSS

CSS means cascading style sheet.

It is used to style an HTML element. It describes how an html page should be displayed.

You have different(three) kinds of way to style an HTML page.

Inline styling, Internal styling and external styling



Inline styling

I am a text styled using an inline styling.

An inline styling is done when you pass a 'style' attribute into the html element and start passing the styles into it.

Internal styling

I am a text styled using an internal styling through a class.

I am a text styled using an internal styling through an id.

An internal styling is done by passing all of your styles inside a 'style' tag inside the 'head' element. You must give the element you wish to style either a class or an ID

External styling

I am a text styled using an external styling through a class.

I am a text styled using an external styling through an id.

An external styling is also like an internal styling but here, all your styles will be in an external css sheet(file). You must give the element you wish to style either a class or an ID
